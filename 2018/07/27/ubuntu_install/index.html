Ubuntu 18.04 LTS安装


闲来无事，折腾一下Ubuntu。在此之前，有过安装archlinux，kali的经验。事实上，物理机安装linux发行版，从零开始，对linux目录结构和磁盘管理等方面的知识，都是一次挑战 @_@，但在此之前，一定要做好功课！

安装时间：2018.6.27

首先，get到Ubuntu镜像

Ubuntu desktop下载，建议采用 torrent+迅雷 的下载方式：>>> BitTorrent

vmware安装iso

  • 虚拟机配置
    1. 客户机操作系统选择：linux >>> Ubuntu 64位
    2. 虚拟机名称：对应用户名，自己修改，这里我改成 Aurora
    3. 位置：建议不放在c盘，在其他盘建议一个目录专门存放虚拟机文件
    4. 后面一路回车
    5. 内存：看实体机，内存大的建议给到3g左右。我的是8g，给了2.5g，因为2g有点卡啊！！
    6. 磁盘：大小30g吧，因为本着折腾的目的，可能会大量安装乱七八糟的软件 >>> 将磁盘存储为单个文件
  • Ubuntu安装
    1. 虚拟机配置结束后，进入虚拟机设置里的CD/DVD,添加下载好的iso文件
    2. 欢迎界面：选择 中文 ，点击安装Ubuntu
    3. 键盘布局：默认
    4. 更新和其他软件：建议最小安装，需要再去自己折腾
    5. 安装类型：清除整个磁盘并安装
    6. 后面回车就行了
  • 主题安装
    1. 安装优化工具：gnome-tweak-tool
    2. 终端命令：sudo apt-get install gnome-tweak-tool
    3. 安装numix：个人感觉很好看，可以百度看一看
    4. 终端命令：sudo add-apt-repository ppa:numix/ppa
    5. 安装主题：numix-gtk-theme
    6. 终端命令：sudo apt-get install numix-gtk-theme numix-icon-theme-circle
    7. 然后打开tweak优化工具设置下载的主题Numix主题，图标设置为Numix-circle
